# Welcome to the GTM Kit Repository

The goal of GTM Kit is to provide a flexible tool for generating the data layer for Google Tag Manager.

## Development

- **Tests:** see [tests/README.md](tests/README.md) for prerequisites, one-command setup, and how to run each harness (PHPUnit unit, PHPUnit integration, Vitest).
